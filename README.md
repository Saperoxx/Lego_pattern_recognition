# Lego_pattern_recognition
This project is for recognizing colors and special patterns of Lego Blocks. Mainly program is made by Numpy and OpenCV library.

Program is recognizing 5 patterns and 6 colors:
### Colors:

-white,

-yellow,

-red,

-green,

-blue,


### Patterns:

-square,

-rectangle,

-tetris shape,

-Z shape,

-L shape,
